---Instruções sobre o Programa-----

Esse programa lê um arquivo .txt de entrada contendo o catálogo com os filmes. O nome do arquivo é filmes.txt

Instruções sobre como utilizar o Menu:
Basta selecionar uma das seguintes opções:

1. Inserir novo filme no catálogo.
2. Remover filme do catálogo.
3. Buscar filme no catálogo.
4. Editar informações sobre um filme. 
5. Exibir todos os filmes do catálogo. 
6. Exibir o filme com maior nota. 
7. Sair.

Para compilar:
Digite make no terminal

Para executar:
./programa

---Informações sobre o Programa-----

Esse exercício foi feito usando o Visual Studio Code para a edição dos arquivos e foi compilado em Linux (Ubuntu- 18.04).

O arquivo filmes.txt contém o catálogo de filmes que eu usei, porém pode ser usado um outro arquivo próprio de quem for utilizar o programa. Basta que o noem também seja filmes.txt e que as informações sobre cada filme sejam separadas apenas por uma vérgula e cada filme estar em uma linha diferente.
