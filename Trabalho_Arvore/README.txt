---Instruções sobre o Programa-----

Esse programa gerencia dados de pacientes de um hospital/clínica.

Instruções sobre como utilizar o Menu:
Basta digitar uma das seguintes opções no terminal:

1. Inserir um novo paciente.
    1.Inserir um paciente (apenas nome).
    2.Inserir um paciente + seu plano de saúde.
    3.Inserir um paciente + o exame a ser agendado.
    4. Voltar ao Menu.
2. Busca um paciente a partir do nome.
3. Exibe todos os pacientes cadastrados.
4.Sair.


Para compilar:
Digite make no terminal

Para executar:
./programa

---Informações sobre o Programa-----

Esse exercício foi feito usando o Visual Studio Code para a edição dos arquivos e foi compilado em Linux (Ubuntu- 18.04).
